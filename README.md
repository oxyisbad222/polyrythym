# PolyRhythm

PolyRhythm is a web and mobile rhythm game inspired by titles like Rockband and Guitar Hero. Play along to your favorite songs using your keyboard on the web or touch screen on your mobile device.

## 🎵 Features

-   **Cross-Platform:** Play on any modern web browser or mobile device.
-   **Custom Songs:** Load your own songs from folders containing `album.jpg`, `.opus` audio files, `notes.mid`, and `song.ini`.
-   **6-Key Gameplay:** A challenging and fun 6-key setup for a fresh rhythm game experience.
-   **Rockband-style Gameplay:** A familiar and intuitive interface for rhythm game enthusiasts.

## Controls

-   **Web:** Use the `A`, `S`, `D`, `J`, `K`, `L` keys to hit the notes.
-   **Mobile:** Tap the corresponding columns on the touch screen to hit the notes.

## 🚀 Building Locally

This project is built with Unity. To get started, you will need to have Unity Hub and a recent version of the Unity Editor installed.

1.  Clone this repository to your local machine.
2.  Open the project in the Unity Editor.
3.  Load your custom songs into the `Songs` directory.
4.  Press the Play button in the Unity Editor to start the game.
